# AutoShop Telegram Mini App

Простое веб-приложение для Telegram бота магазина автозапчастей.

## Установка

1. Склонируйте репозиторий
2. Включите GitHub Pages в настройках репозитория
3. Используйте полученный URL в BotFather для регистрации Mini App

## Использование

Приложение автоматически интегрируется с Telegram и использует его темы оформления.
